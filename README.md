# Employee Management ERP

A web-based **Employee Management System** developed using **ASP.NET Web Forms, C#, ADO.NET, and SQL Server**.
This project helps manage employee records efficiently with features like employee registration, update, delete, search, admin login, and employee data management.

## Features

* Admin login system
* Add new employee details
* View employee records
* Update employee information
* Delete employee records
* Search employee data
* SQL Server database integration
* Clean and simple web interface
* CRUD operations using ASP.NET and C#

## Technologies Used

* ASP.NET Web Forms
* C#
* ADO.NET
* SQL Server
* HTML
* CSS
* Bootstrap
* Visual Studio

## Project Modules

### Admin Module

The admin can log in securely and manage the employee records.

### Employee Module

The employee module allows adding, viewing, updating, deleting, and searching employee details.

### Database Module

Employee data is stored and managed using SQL Server.

## How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/Shivamsingh2003-tech/Employee-Management-ERP.git
```

2. Open the project in **Visual Studio**.

3. Restore required packages if needed.

4. Open SQL Server and create/import the required database.

5. Update the database connection string in `Web.config`.

6. Run the project using Visual Studio.

## Purpose of the Project

The purpose of this project is to provide a simple ERP-style employee management system for maintaining employee records digitally. It is useful for small organizations, academic projects, and learning ASP.NET with SQL Server.
